# 스토리 인사이트 랩

## 소개
스토리 인사이트 랩은 이야기의 구조를 분석하고, 독자의 반응을 예측하며, 살아 있는 캐릭터를 설계하기 위한 창작 지원 도구입니다.

작품의 설정, 줄거리, 인물 관계, 장면 흐름을 정리하고 분석하여 더 설득력 있는 이야기와 매력적인 캐릭터를 만드는 데 도움을 줍니다.

## 주요 기능
- 이야기 구조 분석
- 독자 반응 예측
- 캐릭터 설정 및 관계 분석
- 장면별 감정선 정리
- 설정 오류 및 개연성 점검
- 웹소설, 웹툰, 드라마, 영화 기획 보조

## 사용 방법
1. 프로젝트를 다운로드합니다.
2. 분석하고 싶은 이야기, 캐릭터, 시놉시스, 원고를 입력합니다.
3. 필요한 기능을 선택해 결과를 확인합니다.
4. 분석 결과를 바탕으로 이야기를 수정하거나 확장합니다.

## 활용 예시
- 웹소설 시놉시스의 재미 요소 점검
- 독자가 지루해할 수 있는 구간 예측
- 캐릭터의 욕망, 약점, 관계성 정리
- 장면의 갈등 구조와 감정선 강화
- 작품의 장르성과 시장성 검토
